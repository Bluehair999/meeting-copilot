# MEETING Copilot

AI(Whisper + GPT-4o)를 활용한 실시간 음성 인식 및 회의록 자동 요약 웹 애플리케이션입니다.

## 🚀 동료에게 배포 / 공유하는 방법

이 폴더(`260320 Meeting Copilet`) 자체를 압축해서 동료에게 전달하시되, 용량이 너무 커지는 것을 막기 위해 **아래 두 폴더는 지우고 바탕화면만 깔끔하게 압축해서 보내시는 것**을 권장합니다.
* `backend/venv` 폴더 (파이썬 라이브러리 - 약 500MB)
* `frontend/node_modules` 폴더 (노드 라이브러리 - 약 250MB)

## 💻 새로운 PC(동료)에서 처음 시작하는 방법

동료가 압축을 풀고 프로그램을 처음 실행하려면 다음 조건이 필요합니다.
1. **필수 설치 프로그램:** 해당 PC에 `Python(파이썬)`과 `Node.js(노드JS)`가 설치되어 있어야 합니다.
2. 새 폴더에서 **`3_신규_PC_최초_설치_자동화.bat`** 파일을 더블클릭합니다.
   - 알아서 백엔드와 프론트엔드에 필요한 라이브러리를 몇 분 동안 다운로드하고 설치해 줍니다.
3. 설치가 끝나면, 앞으로는 **`1_Start_Servers.bat`** 파일을 더블클릭하여 서버를 바로 구동할 수 있습니다.

## 🔐 API 키 설정
서버가 켜지고 `http://localhost:5173` 으로 접속했다면, 우측 상단 메뉴에 각 사용자의 **개별 OpenAI API Key**를 입력하고 [Confirm]을 눌러야 AI 기능이 정상적으로 활성화됩니다. (절대로 이 폴더에 내 개인 API 키를 하드코딩해서 공유하지 마세요.)

---
**기능 요약:**
* **실시간 STT:** 마이크 및 시스템 소리를 5초 단위로 캡처하여 Whisper 모델을 통해 자동 받아쓰기
* **실시간 번역:** 입력 언어(한국어/영어/폴란드어)를 타겟 언어로 실시간 GPT 고정 번역
* **AI 회의록 요약:** 회의가 끝난 뒤 `[회의 개요]`, `[주요 안건]`, `[세부 논의 내용]`의 프리미엄 A4 양식으로 자동 텍스트 요약
* **Word 추출:** 작성된 회의록을 깔끔한 형태의 MS Word(.docx) 파일로 다운로드
